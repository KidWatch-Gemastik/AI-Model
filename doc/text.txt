1. Buat virtual environment agar dependencies rapi:
python -m venv venv
source venv/bin/activate  # Linux/macOS
venv\Scripts\activate     # Windows

2. Install package:
pip install fastapi uvicorn easyocr transformers torch pillow
[
    fastapi → buat API endpoint notifikasi

    uvicorn → server FastAPI

    easyocr → OCR

    transformers + torch → NLP model

    pillow → manipulasi gambar (optional)
]


# Running server
python main.py
